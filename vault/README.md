# Vault de firmware

Guardar aca compilaciones UF2 que fueron probadas y funcionaron.

Formato sugerido:

```text
vault/YYYY-MM-DD-descripcion/
  keyball61_right-nice_nano_v2-zmk.uf2
  keyball61_left-nice_nano_v2-zmk.uf2
  README.md
```

En el README de cada build anotar:

- commit usado;
- fecha;
- que cambios incluye;
- que lado se flasheo;
- que pruebas pasaron;
- problemas conocidos.
